Welcome to Plato Tipico


Español
Plato Típico es una distribución basada en pressflow.

Esta distribución nace por la necesidad de optimizar el proceso de arranque de nuevos proyectos, cada vez que se inicia un proyecto se repiten una y otra vez los pasos típicos de instalación y configuración. La idea es que usando la distribución Plato Típico se acelere la productividad dando por sentado el proceso inicial de configuración.

La distribución viene en 2 presentaciones:

Liviano:

Sistema pre configurado con soporte a multi-idioma inglés y español, soporte a localización automática,
WYSIWYG utilizando TinyMCE con soporte a español y algunas características de uso
fundamental en cualquier sitio Drupal.

Si eres fanático de Github puedes descargarlo en https://github.com/enzolutions/plato_tipico


Familiar:

Sistema con todas las características de un plato típico liviano pero además cuenta con algunos módulos de uso frecuente; permitiendo que el administrador pueda integrarlos de forma ágil y rápida en su proyecto.

Dado que Drupal.org no permite subir 2 presentaciones de un mismo módulo, puedes descargarlo de github https://github.com/enzolutions/plato_tipico_familiar.

English
"Plato Típico" is a distribution based on pressflow.

This distribution comes from the need to optimize the process of getting started new projects, each time you beging a project there are always some things that you have to do over and over again, those typical steps during the installation and configuration. The idea is that using the distribution "Plato Tipico" you can improve productivity assuming the initial configuration process.

The distribution comes in two forms:

Light:

Preconfigured system with multi-language support for English and Spanish, supports automatic localization, WYSIWYG using TinyMCE with support for Spanish, and basic features required for any website.

If you're a fan of Github you can download it at https://github.com/enzolutions/plato_tipico


Family:

System with all the features of a light "Plato Tipico." Also includes frequently used modules that can be flexibily integrated in your project.

Being that Drupal.org doesn't allow submission of two different implementations, we are hosting this version on github: https://github.com/enzolutions/plato_tipico_familiar.

This distribution is developed and maintained by: Anexus IT http://anexusit.com


INSTALLATION

Those familiar with drush make can find a build-openpublic.make file available
in this folder that can be used to build OpenPublic with a command such as:

drush make --prepare-install build-plato_tipico.make INSTALL_PATH

Please consult:
  http://drupalcode.org/project/drush_make.git/blob_plain/refs/heads/6.x-2.x:/README.txt
for more information about using this.
